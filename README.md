# Zilk

Zilk is a simple single-page ecommerce marketplace website designed for buying brand new and used tech products in a clean and modern way.

The website showcases real products such as Apple MacBook, ASUS ZenBook, Apple AirPods, Sony earbuds, and a Razer mouse. It is designed with a white and light yellow color theme to give the page a soft, clean, and modern shopping feel.

The homepage includes a featured hero section, product cards with matching real product images, hover effects, smooth scroll animations, customer reviews from Ankita, Jaimin, and Sakib, and a simple shopping experience with add to cart functionality.

Zilk also includes a checkout section where users can enter their order details, along with a contact section for sending customer messages. Everything is kept in a very simple structure using HTML, CSS, and JavaScript.
